# Ad Platform Specifications

Hard character limits and format requirements. Reference during copy writing — every field must be within spec before delivery.

---

## Meta (Facebook + Instagram)

### Single Image / Video Ad

| Field | Limit | Notes |
|-------|-------|-------|
| Primary text | 2,200 chars | 125 chars visible before "see more" — hook must land here |
| Headline | 40 chars | Shown below image/video |
| Description | 30 chars | Shown under headline on some placements |
| Link URL | — | Must be UTM-tagged |

**Recommended length:** Primary text 100–150 chars for highest reach and engagement. Longer copy works but reduces delivery on some placements.

### Carousel Ad

| Field | Limit | Notes |
|-------|-------|-------|
| Primary text | 2,200 chars (shared) | Same 125-char visible rule |
| Headline per card | 40 chars | Each card has its own headline |
| Description per card | 30 chars | Each card has its own description |
| Cards | 2–10 | First card must be strongest |

### Stories Ad

| Field | Limit | Notes |
|-------|-------|-------|
| Primary text overlay | 90 chars | Text on screen within safe zone |
| Headline | 40 chars | Shown in some placements |

**Stories note:** Copy is largely on-screen text overlaid on creative. Keep overlays minimal — the visual does more work here.

### Lead Ad

| Field | Limit | Notes |
|-------|-------|-------|
| Primary text | 2,200 chars | Same 125-char visible rule |
| Headline | 40 chars | |
| Description | 30 chars | |
| Form intro headline | 60 chars | Shown at top of lead form |
| Form intro body | 500 chars | Explains what they're signing up for |
| Thank you headline | 60 chars | Post-submission message |

## Google Ads

### Responsive Search Ad (RSA)

| Field | Limit | Per Ad | Notes |
|-------|-------|--------|-------|
| Headline | 30 chars | 3 shown / 15 max | Google tests combinations; write 10–15 |
| Description | 90 chars | 2 shown / 4 max | Write all 4; Google tests combinations |
| Path 1 | 15 chars | 1 | Shown in URL — keyword relevant |
| Path 2 | 15 chars | 1 | Shown in URL — more specific |
| Final URL | — | 1 | UTM-tagged destination URL |

**Headline diversity requirement:** Headlines must not be variations of each other — Google penalizes duplicate themes. Include:
- Keyword headlines (contain the target keyword)
- Benefit headlines (what they get)
- CTA headlines ("Get a Quote", "Start Free Trial")
- Differentiator headlines ("No Setup Fees", "Cancel Anytime")
- Social proof headlines ("Rated #1 by...", "Join 10,000+")

**Pinning:** Optional. Pin a headline to Position 1, 2, or 3 when a specific message must always appear (brand name, required disclosure). Use sparingly — pinning reduces combinatorial testing.

## TikTok Ads

### In-Feed Ad

| Field | Limit | Notes |
|-------|-------|-------|
| Ad name (internal) | 512 chars | Not shown to users |
| Caption | 2,200 chars | First 100 chars shown; emoji supported |
| Display name | 20 chars | Brand name in the ad |
| CTA | Select from list | Options: Learn More, Shop Now, Sign Up, Download, Contact Us, Order Now, Book Now, Subscribe, Apply Now, Get Quote, Watch More, Listen Now |

**TikTok copy philosophy:** The caption is secondary to the video. On-screen text, voiceover, and the hook in the first 1–3 seconds of video are where conversion happens. Write the caption to reinforce and extend the video — not to lead it.

**For the video itself, provide:**
- Hook script (first 3 seconds — exact words spoken or shown on screen)
- On-screen text supers (key lines to display as text overlays)
- Full voiceover/dialogue script if applicable

### TopView Ad
Same specs as In-Feed. Appears as first video in feed — hook in the first second is critical.

### Branded Hashtag Challenge
- Challenge name: 100 chars
- Description: 150 chars
- Landing page copy: standard landing page rules apply

## LinkedIn Ads

### Single Image Ad

| Field | Limit | Notes |
|-------|-------|-------|
| Introductory text | 600 chars | 150 chars visible before "see more" — hook here |
| Headline | 200 chars | Shown below image |
| Description | 300 chars | Shown under headline in some placements |

**LinkedIn copy note:** Professional audience; B2B skew. Specificity and business relevance outperform consumer emotional hooks. "Reduce your engineering team's deploy time by 40%" beats "Work faster, ship more." Name the business outcome.

## Pinterest Ads

### Promoted Pin (Static)

| Field | Limit | Notes |
|-------|-------|-------|
| Title | 100 chars | Descriptive; keyword-rich |
| Description | 500 chars | Discoverable in search; tell the story |

**Pinterest copy philosophy:** Pinterest is part search engine, part inspiration board. Keywords matter for discovery — include them naturally in the description. Descriptions that tell a mini-story or provide genuine value perform better than pure promotional copy.

## Hook Formulas Quick Reference

Build the hook first. Then write everything else.

**Pattern interrupt:**
"Most [category] advice is wrong."
"[Common belief] is a myth."
"[Expected statement] — except when [subversion]."

**Problem precision:**
"If you're [specific situation], this is for you."
"[Very specific frustration]? Here's why."
"The [timeframe] I spent [painful activity] before I found this."

**Specific claim:**
"[Specific result] in [specific timeframe]."
"[Number] customers. [Specific outcome]. [How]."
"We [specific action] so you don't have to [pain]."

**Curiosity:**
"The one thing [relevant expert] does differently."
"What nobody tells you about [category]."
"[Question that assumes the reader has a desire they haven't voiced]?"

**Before/after:**
"Before: [pain state]. After: [desired state]. Here's what changed."
"[Company/person] went from [before] to [after] using [mechanism]."

**Contrarian:**
"Stop [common behavior everyone does]."
"[Popular advice] is destroying your [outcome]."
"We don't [expected thing]. Here's why."

## CTA Button Copy Reference

Write CTAs that say what happens when the reader clicks — not generic filler.

**Strong CTA patterns:**
- "Start my free trial" (not "Start a free trial")
- "Get [specific thing]" — "Get the guide", "Get your score", "Get my discount"
- "See how it works"
- "Claim my spot" (urgency + possession)
- "Yes, I want [benefit]"
- "[Action] now" — "Book now", "Shop now", "Apply now"

**Weak CTA patterns to avoid:**
- "Click here"
- "Submit"
- "Learn more" (acceptable but weak — only use when the action is genuinely exploratory)
- "Get started" (too vague — started on what?)
