# JTC Plugins Marketplace

Curated plugin marketplace for **JTC Enterprises** — built by Strategy Labs.

## Installation

1. Open Claude Desktop (Cowork mode)
2. Add this marketplace:
   ```
   /plugin marketplace add Ramsey-SL/jtc-plugins
   ```
3. Install any plugin:
   ```
   /plugin install ad-copy@jtc-plugins
   /plugin install brand-intelligence-center@jtc-plugins
   /plugin install jtc-social-media-writer@jtc-plugins
   ```
   (Replace the plugin name with whichever you want to install.)

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **ad-copy** | Writes paid ad copy for Meta, Google, TikTok, LinkedIn, Pinterest |
| **brand-asset-manager** | Organizes and catalogs brand photos and videos |
| **brand-intelligence-center** | Builds comprehensive brand profiles with guided setup |
| **brand-knowledge-center** | Brand knowledge management for business owners |
| **competitive-landscape** | Competitor tracking, ad analysis, and journey mapping |
| **content-library** | Stores and retrieves approved copy and visual assets |
| **customer-driven-optimizations** | Analyzes customer calls for insights and site optimization |
| **inbox-command-center** | Email and messaging management across platforms |
| **jtc-social-media-writer** | JTC-specific social media post generator (Facebook, Instagram, Google Business) |
| **lead-list-builder** | Builds targeted lead lists with company and contact data |
